지역별 확진자 비교 통계

특정 지역의 감염 양상을 보여줍니다.
지역별 관리 실태 + 해당 지역의 방역이 원활하게 진행되고 있는지 정도를 파악할 수 있습니다.


사용방법

1. 비교할 지역을 선택하고 '도시 추가하기'를 눌러주세요'
2. (1) 을 필요한 만큼 반복해 주세요
3. 데이터 조회의 시작점이 되는 [연-월-일]을 지정해 주세요
4. 조회할 기간을 설정하고 입력 버튼을 눌러주세요.
